
## Prepare

run to install package: [python3-aiohttp](https://packages.ubuntu.com/bionic/python3-aiohttp)

``` sh
$ sudo apt-get install python3-aiohttp
```


## Test

> Please replace [main.sh / [{YOUR_API_KEY}](main.sh#L9)] first!

Then

run

``` sh
$ ./__main__.py
```
